## Hugo Fullstack API

## Requirements

- Node v18
- pnpm

> Note: if you don't have pnpm, you will need to install it

```bash
npm i -g pnpm
```

## Getting Started

First, run the development server:

```bash
pnpm install
```

By default, the application will run at [http://localhost:3001](http://localhost:3001).

## Dev

### Start

```bash
pnpm dev
```

### Test (watch)

```bash
pnpm test:watch
```

## Production

### Test

```bash
pnpm test
```

### Build

```bash
pnpm build
```

### Start

```bash
pnpm start
```
